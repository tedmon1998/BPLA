# Инструкция пользователя: `perexvat_signala`

## Кнопки и когда они появляются

### 1) Стартовый экран
- **`Активировать`** — видна сразу после запуска.
  - Открывает основной экран при верном коде.
  - При ошибке показывает `Неверный код активации.`

### 2) Основной экран (после активации)
- **`Закончить попытку`** — видна всегда во время сессии.
  - Завершает попытку, сбрасывает данные, возвращает на экран активации.

- **`SCAN` / `TRACK` / `RELAY` / `GUARD`** — видны всегда на пульте.
  - Переключают режим связи (активная кнопка подсвечивается).

- **`Отправить координаты`** — появляется в блоке `Проверка координат`.
  - Доступна только при стабильном сигнале.
  - Если сигнал не настроен: `Нет сигнала. Сначала настройте радиомикшер.`
  - Если формат координат неверный: сброс попытки.
  - Если координаты верные: открывается окно с ключом.

### 3) Окно подтверждения
- **`Закрыть`** — появляется после успешной проверки координат.
  - Закрывает окно и сбрасывает раунд.
  - Без нажатия окно закроется само через 10 секунд.

## Важные условия
- Для стабильного сигнала настройте: `FREQUENCY`, `CHANNEL`, `GAIN`.
- Блок `Диагностика настройки` подсказывает, какой регулятор мешает захвату:
  - частота: `близко` / `далеко`;
  - канал: `нестабилен`;
  - усиление: `недостаточно`.
- Точные целевые значения не показываются.
- Поля `Lat/Lon` и кнопка отправки работают только при стабильном сигнале.
- Формат координат: десятичные градусы, ровно 2 знака после точки (например, `55.75`).

## Задание
После восстановления канала нужно найти **ближайшую точку конфлюэнции** и ввести её координаты.

## Краткий порядок действий
1. Введите код и нажмите `Активировать`.
2. Настройте сигнал до статуса `LINK SECURED`, ориентируясь на диагностику.
3. Найдите ближайшую точку конфлюэнции.
4. Введите её координаты `Lat` и `Lon`.
5. Нажмите `Отправить координаты`.
6. Получите ключ и нажмите `Закрыть` (или дождитесь автозакрытия).
