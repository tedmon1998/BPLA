# Инструкция пользователя (ai_praktikum)

## 1. Экран активации

### Кнопка `Войти`
- **Где:** окно `Активация`.
- **Когда видна:** до входа в практикум.
- **Когда активна:** всегда, пока не идет проверка.
- **Что делает:** проверяет код активации и открывает уровни.
- **Особенность:** во время проверки текст меняется на `Проверка...`, кнопка временно блокируется.

---

## 2. Верхняя панель после активации

### Ссылка с названием приложения
- **Где:** слева в шапке.
- **Когда видна:** всегда после активации.
- **Что делает:** возвращает на главный экран курса.

### Кнопка `Закончить попытку`
- **Где:** справа в шапке, рядом с таймером `Сессия`.
- **Когда видна:** во время активной сессии.
- **Что делает:** завершает текущую попытку и возвращает на экран активации.

---

## 3. Кнопки навигации по уровням

### Кнопка `Назад`
- **Когда видна:** внизу каждого уровня.
- **Когда активна:** со 2-го уровня и дальше.
- **Что делает:** возвращает на предыдущий уровень.

### Кнопка `Уровень пройден, далее`
- **Когда видна:** на уровнях 1-5.
- **Когда активна:** только после успешного выполнения текущего уровня.
- **Что делает:** переводит на следующий уровень.

### Кнопка `Завершить курс`
- **Когда видна:** на последнем уровне вместо кнопки `Уровень пройден, далее`.
- **Когда активна:** только после успешной проверки на последнем уровне.
- **Что делает:** открывает окно награды с кодом.

### Кнопка `Закрыть` (в окне награды)
- **Когда видна:** после завершения всех уровней.
- **Что делает:** закрывает окно награды и перезапускает курс с 1 уровня.
- **Особенность:** если не нажимать, окно закроется автоматически по таймеру.

---

## 4. Кнопки внутри уровней

### Уровень `Логический нейрон`
- `Сброс параметров` - сбрасывает `W1`, `W2`, `θ` к исходным значениям.

### Уровень `Логическая нейросеть`
- `Обучить сеть` - запускает обучение с текущими параметрами.

### Уровень `Четные / нечетные`
- Кнопки-классы `Четное` / `Нечетное` - выбирают целевой класс примера.
- Клетки сетки - включают/выключают пиксели образа.
- `Добавить пример` - добавляет текущий образ в выборку.
- `Обучить` - обучает модель.
- `Классифицировать` - проверяет текущий образ.
- `Удалить` (у конкретного примера) - удаляет этот пример.
- `Удалить все примеры` - очищает весь набор примеров.
- `Очистить сетку` - очищает только текущий рисунок.

### Уровень `Буквы A/B/C`
- Кнопки-классы `A`, `B`, `C` - выбирают целевой класс.
- Клетки сетки - рисование образа.
- `Добавить пример`, `Обучить`, `Классифицировать`, `Удалить`, `Удалить все примеры`, `Очистить сетку` - работают аналогично уровню выше.

### Уровень `Диагноз: грипп`
- Чекбоксы симптомов - формируют входные признаки.
- Чекбокс `Болен гриппом` - задает целевой класс (`болен` / `не болен`) для добавляемого примера.
- `Добавить пример` - добавляет пациента в выборку.
- `Обучить` - обучает модель.
- `Поставить диагноз` - рассчитывает диагноз для текущего набора симптомов.
- `Очистить вход` - снимает выбранные симптомы.
- `Удалить` (в списке примеров) - удаляет конкретный пример.

### Уровень `Несколько диагнозов`
- Кнопки-классы `Пневмония`, `ОРЗ`, `Здоров` - задают целевой класс.
- Чекбоксы симптомов - формируют вход.
- `Добавить пример` - добавляет пример в выборку.
- `Обучить` - обучает многоклассовую модель.
- `Рассчитать` - определяет диагноз для текущего входа.
- `Очистить вход` - очищает выбранные симптомы.

---

## 5. Когда кнопки не срабатывают

- `Уровень пройден, далее` и `Завершить курс` недоступны, пока текущий уровень не выполнен успешно.
- Кнопки обучения (`Обучить`, `Обучить сеть`) выводят сообщение, если примеров недостаточно или не представлены все требуемые классы.
- Кнопки проверки (`Классифицировать`, `Поставить диагноз`, `Рассчитать`) требуют, чтобы модель уже была обучена.
- На первом уровне кнопка `Назад` неактивна.
