# Нейросетевой практикум (SPA)

Учебное веб-приложение на `React + TypeScript` для моделирования логических функций, обучения перцептронов и простой нейросети, а также визуализации результатов.

## Статус реализации

Текущая версия содержит:

- базовый SPA-каркас и маршрутизацию всех модулей из ТЗ;
- полностью рабочий экран `Логические функции нейрона` (AND/OR/NAND/NOR/XOR);
- доменный слой для порогового нейрона;
- тесты бизнес-логики для модуля логического нейрона;
- заготовки остальных экранов для поэтапной реализации MVP.

## Запуск проекта

```bash
npm install
npm run dev
```

Приложение запускается по адресу, который выводит Vite (обычно `http://localhost:5173`).

## npm-скрипты

- `npm run dev` — запуск в режиме разработки;
- `npm run build` — production-сборка;
- `npm run preview` — просмотр production-сборки;
- `npm run lint` — проверка ESLint;
- `npm run test` — запуск unit-тестов;
- `npm run test:watch` — тесты в watch-режиме;
- `npm run coverage` — отчет по покрытию тестами.

## Архитектура

Проект организован по слоям:

- `src/app` — роутинг и layout приложения;
- `src/pages` — экраны (модули);
- `src/domain` — математическая и алгоритмическая логика;
- `src/shared` — роуты, словари, общие утилиты;
- `src/test` — тестовая конфигурация.

## Инструкция пользователя (текущий модуль)

### Логические функции нейрона

1. Перейдите в модуль `Логические функции нейрона`.
2. Выберите функцию (`AND`, `OR`, `NAND`, `NOR`, `XOR`).
3. Меняйте параметры `W1`, `W2`, `θ`.
4. Смотрите обновление таблицы истинности и протокола в реальном времени.
5. При полном совпадении `y` с `d` модуль покажет успешное моделирование.
6. Для `XOR` выводится сообщение о линейной неразделимости для одиночного нейрона.

## Ограничения и допущения (текущая версия)

- Реализован только первый модуль из ТЗ, остальные представлены каркасами.
- Визуализация 2D/3D пока не добавлена.
