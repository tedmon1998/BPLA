# Инструкция пользователя (analiz_telemetrii)

## 1. Экран активации

### Поле `Код активации`
- **Когда доступно:** сразу после запуска приложения.
- **Что делать:** введите выданный код.

### Кнопка `Активировать`
- **Когда видна:** на экране активации.
- **Когда активна:** если поле кода не пустое и не идет обработка запроса.
- **Что делает:** запускает попытку и открывает рабочий экран.
- **Если ошибка:** отображается сообщение (например, неверный код или превышен лимит попыток).

---

## 2. Основной экран

### Кнопка `Закончить попытку`
- **Когда видна:** только при активной попытке.
- **Что делает:** немедленно завершает попытку.
- **Результат:** возврат на экран активации, очистка данных ввода для текущего раунда.

### Поле `Код страны (ISO 3166-1 alpha-2)`
- **Когда доступно:** только при активной попытке.
- **Что делать:** введите код страны (обычно 2 латинские буквы, например `RU`, `KZ`, `US`).
- **Важно:** без кода страны проверка клика по графику не выполняется.

### Клик по графику
- **Когда доступен:** при активной попытке и загруженных данных.
- **Где работает:** на графиках телеметрии.
- **Что делает:** отправляет на проверку выбранный момент времени вместе с кодом страны.
- **Результат:** при успехе открывается окно результата, при ошибке показывается подсказка.

### Подсказка на графике (tooltip)
- **Когда появляется:** при наведении курсора на точку графика.
- **Что показывает:** `time`, `height`, `speed`, `battery`, `voltage`, `lat`, `lon`.
- **Зачем нужна:** помогает выбрать корректный момент аномалии и определить страну по координатам.

---

## 3. Окно результата

### Кнопка `Закрыть`
- **Когда видна:** после успешной проверки, в модальном окне.
- **Что делает:** закрывает окно результата и запускает новый раунд.
- **Результат:** поле кода страны очищается, попытка продолжается.

### Автозакрытие
- **Когда срабатывает:** если кнопку `Закрыть` не нажимать.
- **Что происходит:** по таймеру окно закроется автоматически, затем начнется новый раунд.

---

## 4. Полезные ссылки

- `поиск OpenStreetMap` - поиск страны по координатам.
- `список кодов ISO2` - справочник двубуквенных кодов стран.

---

## 5. Типовые состояния

- `Загрузка телеметрии…` - данные еще загружаются.
- `Сначала активируйте попытку.` - попытка не активирована.
- `Время попытки истекло. Введите код активации снова.` - время закончилось, нужна повторная активация.
- Сообщения вида `wrong_time`, `wrong_country`, `wrong_both` - неверный момент, неверная страна или обе ошибки сразу.
