# Головоломка: телеметрия → страна → код

> **Для участников** раздавайте только блок «Задание». Раздел «Решение» — только для жюри, не публикуйте в открытом репозитории к конкурсу.

---

## Задание для участников

Беспилотник передал телеметрию: время, высота, скорость, заряд, напряжение, а также **широта и долгота (lat, lon)**. В данных есть шум и **ложные просадки** напряжения; есть **ровно одна критическая аномалия питания**.

1. По графикам найдите эту аномалию (в т.ч. по согласованности кривых: напряжение, скорость, заряд, при необходимости высота).
2. В **подсказке** на графике в этой точке посмотрите **lat и lon**.
3. По картам определите, **в какой стране** соответствует эта точка (например, через поиск по координатам на OpenStreetMap).
4. Введите **код страны ISO 3166-1 alpha-2** (две латинские буквы) в форму на странице.
5. **Кликните** по графику в момент аномалии. При верном времени и верной стране вы получите **ключ**.

Никакие файлы подгружать не нужно: данные открываются сразу.

---

## Решение (жюри)

### Демо-данные

- Файл: `public/data/telemetry.csv` (после сборки — `dist/data/telemetry.csv`).
- Критическая аномалия (логика `findAnomaly`) — **в момент времени ~58 с**; в этой строке зафиксированы координаты **35.68920°N, 51.38900°E** (Тегеран, **Иран**).
- Успешная проверка на сервере: **время** в окне ±1 с от центра аномалии **и** `countryIso2` = **IR** (из переменной окружения, не из кода).

### Переменные окружения (сервер)

- `REWARD_CODE` — строка, показываемая в модалке при успехе.
- `EXPECTED_COUNTRY_ISO2` — **ровно две** латинские буквы (например `IR`). **Не** используйте префикс `VITE_`.

### Где в коде

- Проверка: `server/secureApi.mjs` — сравнение времени и ISO2 с `timingSafeEqual` для кода страны.
- Клиент **не содержит** ни страны, ни кода награды — только ввод пользователя и данные из CSV по HTTP.

---

## Безопасность (кратко)

- Ответы **не зашиты** в фронтенд: `REWARD_CODE` и `EXPECTED_COUNTRY_ISO2` только на сервере.
- Ссылки в UI — **общие** (поиск OSM, вики по ISO2), без подставленных координат.
- Сессия + CSRF + лимит запросов; при HTTPS — `COOKIE_SECURE=1`.

---

## Как получить код на стенде

1. `npm run dev` или `npm run build` + `npm run start`.
2. В `.env`: `REWARD_CODE=...`, `EXPECTED_COUNTRY_ISO2=IR` (или другая страна, если замените координаты в CSV и env согласованно).
